Developer Mode Setup

Quick installation for Chrome and Edge browsers.

Google Chrome

  • 01. Navigate to chrome://extensions
  • 02. Enable Developer Mode at top right
  • 03. Click Load Unpacked at top left
  • 04. Select your extension source folder

Microsoft Edge

  • 01. Navigate to edge://extensions
  • 02. Toggle Developer Mode sidebar
  • 03. Select Load Unpacked from toolbar
  • 04. Confirm folder selection for installation
